A simple, elegant company website for a beauty salon built with SvelteKit. Includes pages for Home, Services, Gallery, Pricing, About, and Contact/Booking with a server-side form action.

## Features

- Responsive layout with navigation and footer
- Landing page with hero, services overview, testimonials
- Dedicated pages: Services, Gallery, Pricing, About
- Contact & booking form with basic validation and server action stub

## Developing

Install dependencies and start the dev server:

```sh
npm install
npm run dev -- --open
```

## Building

Create a production build:

```sh
npm run build
```

Preview the production build with:

```sh
npm run preview
```

To deploy, add an appropriate [adapter](https://svelte.dev/docs/kit/adapters) for your target platform.
